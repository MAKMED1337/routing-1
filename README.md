# Routing Algorithm Library

This project implements a C++ routing algorithm library for directed road graphs
imported from OpenStreetMap data.  It includes shortest-path algorithms such as
Dijkstra, A*, Contraction Hierarchies (CH), Arc Flags, CHASE, ALT, Bidirectional
variants, PHAST, and Hub Labels.

I implemented the library of algorithms using AI as the code author.  My role
was to direct the work, review the generated code, validate the implementations,
and benchmark the results.

The report PDF is available at `LaTex/report.pdf`.
